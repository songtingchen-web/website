Executive Summary

This project analyses Nestlé’s corporate governance structure, ESG accountability, climate risk management, and sustainability performance using real-world data from the 2024 Non-Financial Statement, Corporate Governance Report, and public audits.

Using Python, we visualise: - Board composition & independence - ESG-linked executive remuneration - Plastic reduction target performance - Governance accountability gaps

This analysis applies Agency Theory and Stakeholder Theory to evaluate governance effectiveness in delivering genuine climate action.

3. Visualise ESG & Governance Performance

View Python Source Code 
import matplotlib.pyplot as plt
import numpy as np

# Suppress font warnings
plt.rcParams['font.family'] = 'sans-serif'
plt.rcParams['font.sans-serif'] = ['Arial', 'DejaVu Sans', 'sans-serif']

# Prepare Radar Chart data
num_vars = len(metrics)
angles = np.linspace(0, 2 * np.pi, num_vars, endpoint=False).tolist()
values_radar = values + values[:1]  # Close the loop
angles += angles[:1]

# Reduced figsize (5x5) for a more compact radar chart
fig, ax = plt.subplots(figsize=(5, 5), subplot_kw=dict(polar=True))

# Draw the radar plot
ax.fill(angles, values_radar, color='#2E86AB', alpha=0.25)
ax.plot(angles, values_radar, color='#2E86AB', linewidth=1.5, marker='o', markersize=4)

# Professional styling
ax.set_theta_offset(np.pi / 2)
ax.set_theta_direction(-1)
plt.xticks(angles[:-1], metrics, fontsize=8) # Smaller font for cleaner look

ax.set_rlabel_position(0)
plt.yticks([25, 50, 75, 100], ["25%", "50%", "75%", "100%"], color="grey", size=7)
plt.ylim(0, 100)

plt.title('ESG Performance Radar (2024)', fontsize=11, fontweight='bold', pad=20)
plt.tight_layout()
plt.show()

  • Key Insight: Nestlé shows exceptional Board Independence (92%), providing a strong foundation for oversight.

  • Observation: There is a visible skew towards governance structure over environmental target achievement.

5. Plastic Reduction Target Gap Analysis

View Python Source Code 
import matplotlib.pyplot as plt

# Using a much slimmer height (1.6) for a refined dashboard appearance
plt.figure(figsize=(7, 1.6))

# Suppress font warnings
plt.rcParams['font.family'] = 'sans-serif'
plt.rcParams['font.sans-serif'] = ['Arial', 'DejaVu Sans', 'sans-serif']

# Data points
categories = ['Plastic Reduction']
actual_val = 21.3
target_val = 33

# Plot target bar (background) and actual bar (foreground)
plt.barh(categories, [target_val], color='#E0E0E0', label='Target (33%)', height=0.4)
plt.barh(categories, [actual_val], color='#A23B72', label='Actual (21.3%)', height=0.2)

# Add target vertical line
plt.axvline(x=target_val, color='#C73E1D', linestyle='--', linewidth=1.5)

# Add value annotations
plt.text(target_val + 0.5, 0, f'Target: {target_val}%', color='#C73E1D', 
         va='center', fontweight='bold', fontsize=9)
plt.text(actual_val - 0.8, 0, f'{actual_val}%', color='white', 
         va='center', ha='right', fontweight='bold', fontsize=8)

# Styling and minimalist cleanup
plt.title('Progress Toward 2025 Sustainability Goal', fontsize=11, fontweight='bold', pad=10)
plt.xlabel('Reduction Percentage (%)', fontsize=9)
plt.xlim(0, 45)
plt.yticks([]) # Hide y-axis text to save space and look cleaner
plt.legend(loc='lower right', fontsize=8, frameon=False)
plt.grid(axis='x', linestyle=':', alpha=0.3)

plt.tight_layout()
plt.show()

  • Key Insight: An 11.7% gap currently exists between the 2024 performance (21.3%) and the 2025 plastic reduction goal (33%).

  • Strategic Risk: Accelerated investment in circular economy infrastructure is required to mitigate potential greenwashing accusations.

import pandas as pd
import plotly.express as px

df_inc = pd.DataFrame(incentive_data)

fig = px.pie(df_inc, values='Weight (%)', names='Component',
             title='Executive Remuneration Structure',
             hole=0.45,  # Slightly larger hole for a more modern look

                 'Base Salary': '#2E86AB',

             },
             width=550, height=380)

    title_font_size=13,
    title_font_weight='bold',
    margin=dict(t=50, l=10, r=10, b=10),
    plot_bgcolor='#fafafa',
    paper_bgcolor='white',
    legend=dict(orientation="h", yanchor="bottom", y=-0.2, xanchor="center", x=0.5)
)

# Optimize text and hover tooltips
fig.update_traces(
    textposition='inside',
    textinfo='percent',
    hovertemplate='<b>%{label}</b><br>Weight: %{value}%<br>%{customdata}',
    customdata=df_inc['Description']
)

fig.show()

  • Key Insight: 40% of executive variable pay is directly linked to ESG and Climate targets, aligning management incentives with long-term sustainability.

  • Theoretical Alignment: This structure utilizes Agency Theory to ensure executives prioritize environmental outcomes alongside financial returns.

7. Governance Analysis & Professional Insights

Key Findings:

  1. Strong Governance Structure
    • 92% independent directors (12/13) – exceeds industry average of 75%
    • Dedicated Sustainability Committee at board level, with quarterly ESG audits
    • ESG metrics integrated into executive pay (35% of variable remuneration linked to sustainability)
  2. Critical Accountability Gap
    • Only 21.3% achieved against a 33% plastic reduction target (2025 goal)
    • Governance design (policy, structure) exceeds operational delivery (execution, outcomes)
    • Tension between corporate lobbying (e.g., on packaging regulations) and public sustainability commitments
  3. Theoretical Framework
    • Agency Theory: ESG KPIs align management incentives with shareholder climate risk management
    • Stakeholder Theory: Sustainability programs address environmental, social, and regulatory stakeholder demands
    • Hybrid model reveals implementation gaps between policy design and on-the-ground practice

Recommendations:

  • Establish a dedicated Climate Oversight Committee (independent directors only) to monitor target progress
  • Allocate 30% of Capital Expenditure (CapEx) to decarbonisation and circular economy initiatives
  • Mandate board-level climate literacy training (aligned with TCFD recommendations)
  • Implement quarterly public disclosures on plastic reduction progress to enhance transparency

## 7. Governance Analysis & Professional Insights
### Key Findings:
1. **Strong Governance Structure**
   - 92% independent directors (12/13) – exceeds industry average of 75%
   - Dedicated Sustainability Committee at board level, with quarterly ESG audits
   - ESG metrics integrated into executive pay (35% of variable remuneration linked to sustainability)

2. **Critical Accountability Gap**
   - Only **21.3% achieved** against a 33% plastic reduction target (2025 goal)
   - Governance design (policy, structure) exceeds operational delivery (execution, outcomes)
   - Tension between corporate lobbying (e.g., on packaging regulations) and public sustainability commitments

3. **Theoretical Framework**
   - **Agency Theory**: ESG KPIs align management incentives with shareholder climate risk management
   - **Stakeholder Theory**: Sustainability programs address environmental, social, and regulatory stakeholder demands
   - Hybrid model reveals implementation gaps between policy design and on-the-ground practice

### Recommendations:
- Establish a dedicated **Climate Oversight Committee** (independent directors only) to monitor target progress
- Allocate 30% of Capital Expenditure (CapEx) to decarbonisation and circular economy initiatives
- Mandate board-level climate literacy training (aligned with TCFD recommendations)
- Implement quarterly public disclosures on plastic reduction progress to enhance transparency
